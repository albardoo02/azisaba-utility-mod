# Azisaba Utility Mod for 1.15.2

Azisaba Utility Modの一部機能をMinecraft 1.15.2向けに移植したバージョンです．

※ModコマンドはFabric APIの関係上移植していません．

## 前提条件
- Fabric 1.15.2の起動構成があること

- Minecraft 1.15.2向けのFabric APIを使用していること
## 導入方法
1. [Releases](https://github.com/albardoo02/azisaba-utility-mod/new/ver/1.15.2/releases/latest)から最新のファイルをダウンロードする

2. [Fabric API](https://modrinth.com/mod/fabric-api/version/0.28.5+1.15)からFabric API 0.28.5をダウンロードする

3. 起動構成のあるフォルダ直下にあるmodsフォルダにAzisabaUtilityModとFabricAPIを入れて起動する

4. 起動構成のあるフォルダ直下にあるconfigフォルダを開き，`azisabautilitymod.json`にAPIキーを設定する

5. ランチャーを再起動して，動作していることを確認する
   
## APIキーについて

- APIキーはアジ鯖内で`/apikey`をすることで入手可能です．
